# 🕵️‍♂️ 生活尋寶猜謎機 (naiveAI-02) - 親子共戰版

延續 [naiveAI-01](https://github.com/captain-balung/naiveAI-01) 的輕量化與教育科技精神，`naiveAI-02` 是一套專為假日親子互動設計的「反向猜謎」AI 遊戲。

我們將螢幕內的學習轉化為真實世界的探索。透過 Google Apps Script (GAS) 後端串接 LLM 代理端點，打造出一台「聰明又狡猾」的 AI 猜謎機。家長零備課，孩子則能在與 AI 鬥智的過程中，大幅提升觀察力與口語表達能力！

## ✨ 核心亮點

* **反向猜謎，化被動為主動：** AI 出題，孩子給線索！訓練孩子精準描述物品特徵的能力（例如：如何用幾個條件，讓 AI 只能猜出「西瓜」而不是「蘋果」）。
* **狡猾的動態收斂 AI：** AI 被設定為「盡量不輕易認輸」。只要線索不夠明確，AI 就會合理地猜錯，激發孩子的挑戰欲與同儕討論的熱情。
* **極輕量架構，零伺服器成本：** 採用純 HTML/CSS/Vanilla JS 作為前端，搭配 Google Apps Script (GAS) 作為後端 API，免除任何伺服器建置與維護費用。
* **家長零備課負擔：** 一鍵讓 AI 隨機產出符合國小程度的題目，家長只需擔任引導者的角色。

## 🚀 遊戲怎麼玩？（簡單三步驟）

1.  **一鍵出題：** 家長點擊網頁上的「🎲 請 AI 出個題目」。
2.  **保守秘密：** 畫面上會顯示今天的秘密目標（例如：長頸鹿）。家長將答案記在心裡，並把手機交給孩子。
3.  **開始挑戰：** 孩子開始思考秘密目標(如：長頸鹿)的特徵（如：脖子很長、會吃樹葉），並輸入。看看需要幾個回合，才能讓 AI 乖乖認輸說出正確答案！

## 🛠️ 給教育工作者與開發者：如何自行佈署？

如果你喜歡這個專案，並希望擁有一個屬於自己（或自己學校）的專屬猜謎機，只需簡單三個步驟即可完成無痛佈署：

### 步驟一：確認 LLM 代理端點
AI 呼叫已寫在 GAS 後端的 `backend.gs` 中，前端只需要呼叫 GAS Web App URL，不需要放置 Gemini API Key。

### 步驟二：佈署 Google Apps Script (後端大腦)
1. 在你的 Google 雲端硬碟建立一個新的 Google Apps Script 專案。
2. 將本專案中的 `backend.gs` 程式碼完整複製並貼上到你的 GAS 編輯器中。
3. 點擊右上角「佈署」>「新增佈署作業」> 類型選「網頁應用程式」。
4. 「執行身分」選自己，「誰可以存取」**務必選擇「所有人」(Anyone)**。
5. 佈署完成後，複製那串網頁應用程式網址 (Web App URL)。

### 步驟三：修改前端並上線
1. 將本專案 Fork 回你的 GitHub，或是直接下載 `index.html`。
